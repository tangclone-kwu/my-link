# 사용자 시나리오 (User Scenarios)

이 문서는 "마이링크(My Link)" 서비스의 핵심적인 사용 흐름을 방문자(Visitor)와 소유자(Owner) 관점에서 "사용자는 [목적]하기 위해 [행동]한다" 형식으로 구체화한 문서입니다.

---

## 1. 방문자 (Visitor) 시나리오

### 방문자는 소유자의 정보를 한곳에서 확인하기 위해 프로필 페이지에 접속한다.
- 사용자는 인스타그램, 유튜브, 블로그 등 소유자의 다른 채널에 등록된 `[커스텀도메인]/[소유자닉네임]` 형식의 링크를 클릭한다.
- 프로필 페이지가 로드되면 상단의 소유자 닉네임과 간략한 소개글(Bio)을 읽는다.
- 방문자는 목록으로 나열된 소셜 및 포트폴리오 링크 텍스트와 파비콘(아이콘)을 확인한다.
- 자신의 목적에 맞는 특정 버튼(예: "내 포트폴리오 보기")을 클릭하여 해당 외부 웹사이트로 이동한다.

---

## 2. 소유자 (Owner) 시나리오

### 2.1. 계정 생성 및 조회
- 소유자는 자신만의 통합 링크 프로필을 만들기 위해 서비스 홈에서 "Google로 로그인" 버튼을 클릭한다.
- 소유자는 회원가입 절차(추가 정보 입력) 없이 구글 계정 정보를 기반으로 즉시 대시보드에 진입한다.
- 소유자는 자신의 구글 이메일 앞부분(예: `dev123`)이 추출되어 자동으로 생성된 프로필 URL과 닉네임을 대시보드에서 확인한다.

### 2.2. 링크 추가
- 소유자는 새로운 포트폴리오 주소를 프로필에 추가하기 위해 "새 링크 추가" 버튼을 클릭한다.
- 빈 링크 블록이 생성되면, "Title" 영역을 클릭하여 인라인으로 "포트폴리오"라고 입력한다.
- "URL" 영역을 클릭하여 인라인으로 목적지 주소(`https://portfolio.com`)를 입력하고 엔터를 누르거나 바깥 영역을 클릭하여 저장을 완료한다.
- 소유자는 입력한 도메인(`portfolio.com`)의 파비콘이 자동으로 추출되어 링크 블록의 썸네일로 나타남을 확인한다.

### 2.3. 링크 수정
- 소유자는 기존에 등록한 링크의 제목을 변경하기 위해 해당 링크 블록의 "Title" 텍스트를 클릭한다.
- 텍스트가 인라인 입력창으로 변경되면 "이전 포트폴리오"에서 "신규 포트폴리오"로 내용을 수정하고 바깥 영역을 클릭한다.
- 소유자는 프로필 미리보기 화면에서 제목이 즉시 "신규 포트폴리오"로 업데이트된 것을 시각적으로 확인한다.

### 2.4. 링크 삭제
- 소유자는 더 이상 사용하지 않는 외부 링크를 제공하지 않기 위해 해당 링크 블록 우측의 "삭제(휴지통)" 아이콘을 클릭한다.
- 불필요한 알림 모달 없이 즉시(One-click) 삭제 됨을 확인한다.

### 2.5. 프로필(닉네임/소개) 수정
- 소유자는 프로필 URL과 화면 상단에 노출되는 닉네임을 변경하기 위해 프로필 섹션의 기존 닉네임 텍스트를 클릭한다.
- 인라인 입력창에 원하는 새 닉네임(`superdev`)을 입력하고 엔터를 누른다.
- 소유자는 즉시 자신의 프로필 접속 주소가 변경되었음을 인지하고, 방문자들이 보는 닉네임도 변경된 것을 확인한다.

### 2.6. 클립보드 복사 액션 (UX 개선)
- 소유자가 대시보드 상단에 표시된 자신의 프로필 URL 구조(`[도메인]/nickname`) 옆의 '복사' 버튼을 눌렀을 때, 즉시 클립보드에 복사되고 **"복사 완료!"라는 토스트(Toast) 메시지**를 띄워 피드백을 받는다.

### 2.7. 닉네임 자동 중복 처리 로직
- 가입 시 `[구글ID]` 앞자리를 기반으로 자동 닉네임을 생성할 때, 해당 닉네임이 DB의 `users` 서브 컬렉션에 존재한다면 닉네임 뒤에 `-1`, `-2` 같은 **난수(혹은 일련번호)를 자동으로 덧붙여 선점을 우회**한다.
- (예: `dev123`이 있으면 가입 즉시 `dev123-1`을 할당 받고, 추후 인라인 수정을 통해 다른 문자열로 변경한다.)

### 2.8. 빈 링크 방어 로직 (Validation)
- 소유자가 인라인 편집 과정에서 URL을 누락하고 양식을 벗어나거나 올바르지 않은 형식의 문자열을 적었을 경우, 해당 Input 박스의 테두리가 빨간색으로 깜빡이며 사용자의 인지를 돕는다.
